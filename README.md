# A Cripta do Rei Esquecido

## Historia
Voce encontrou uma cripta antiga selada por magia. Simbolos antigos cobrem
as paredes ao redor de uma enorme porta de pedra. Para entrar, voce precisa
decifrar o codigo secreto das runas com base nas pistas reveladas.
Voce tem apenas 3 tentativas antes que a magia te expulse para sempre.

## Como executar
gcc escape-room.c -o cripta && ./cripta

## Exemplo de rodada
=== A Cripta do Rei Esquecido ===
=== AS RUNAS REVELAM ===
- O codigo e um numero PAR
- O codigo nao passa de 30
- O codigo esta entre 10 e 50
Tentativas restantes: 3
Insira o codigo das runas: 20
As runas brilham! A porta de pedra se abre lentamente...
